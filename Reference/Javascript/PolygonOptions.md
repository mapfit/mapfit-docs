# PolygonOptions
### mapfit.PolygonOptions

Inherits from [Polygon](ref:polygon); all methods on the `mapfit.PolygonOptions` class can be called through the `mapfit.Polygon` class.

Method	|Description
--|--
setStrokeColor(color:String)|Sets the polygon outline color to the input color value, which can be specified as an HTML color name (e.g. 'red'), rgb value, or hex value.
setFillColor(color:String)|Sets the polygon color to the input color value, which can be specified as an HTML color name (e.g. 'red'), rgb value, or hex value.
setStrokeWidth(width:Numeric)|Sets the polygon outline stroke width to the input width value, which can be specified as an HTML color name (e.g. 'red'), rgb value, or hex value.

Parameter	|Description
--|--
strokeColor: String|The polygon outline stroke color, if set by the developer. The parameter can be passed in as an HTML color name (e.g. 'red'), rgb value, or hex value.
fillColor: String|The polygon fill color, if set by the developer. The parameter can be passed in as an HTML color name (e.g. 'red'), rgb value, or hex value.
strokeWidth: Integer|The polygon outline stroke width in pixels, if set by the developer. The parameter is passed in as an integer.
