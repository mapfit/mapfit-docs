# PolylineOptions
### mapfit.PolylineOptions

Inherits from [Polyline](ref:polyline); all methods on the `mapfit.PolylineOptions` class can be called through the `mapfit.Polyline` class.
Method	|Description
--|--
setFillColor(color:String)|Sets the polyline color to the input color value, which can be specified as an HTML color name (e.g. 'red'), rgb value, or hex value.
setStrokeColor(color:String)|Sets the polyline outline color to the input color value, which can be specified as an HTML color name (e.g. 'red'), rgb value, or hex value.
setStrokeWidth(width:Numeric)|Sets the polyline outline stroke width to the input width value, which can be specified as an HTML color name (e.g. 'red'), rgb value, or hex value.

Parameter	|Description
--|--
fillColor|The polyline fill color. The parameter is passed in as an HTML color name (e.g. 'red'), rgb value, or hex value.
strokeColor|The polyline stroke color. The parameter is passed in as an HTML color name (e.g. 'red'), rgb value, or hex value.
strokeWidth|The polyline stroke width in pixels. The parameter takes in an integer.
