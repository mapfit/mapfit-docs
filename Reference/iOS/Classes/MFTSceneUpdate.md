## MFTSceneUpdate
`MFTSceneUpdate` initializes a scene update for a specific YAML path in the stylesheet and a new value:

```swift
public class MFTSceneUpdate
```
Property	|Description
--|--
path: String|The YAML path within the stylesheet
Value: String|The value of the new YAML value for the node selected by path.
