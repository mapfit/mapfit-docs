## MFTLineCapType
The shape type for the end of [MFTPolyline](ref:mftpolyline) .

```swift
public enum MFTLineCapType
```
Enum	|Description
--|--
bound|No points added to end of line
square|Two points added to make a square extension
round|Six points added in a fan to make a round cap
