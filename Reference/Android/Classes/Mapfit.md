## Mapfit
### com.mapfit.android.Mapfit

`Mapfit` class is used to initialize the Mapfit Android SDK.

```java
class Mapfit
```

Method	|Description
--|--
getApiKey(): String|Returns the Mapfit API key.
getContext(): Context?|Returns any exceptions association with the Mapfit class intialization.
getInstance(context: Context? = null, apiKey: String): Mapfit|Returns an instance of the Mapfit class for SDK initialization.
