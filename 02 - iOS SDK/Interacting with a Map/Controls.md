# Controls
Below is a list of controls available when using our maps. Toggling these will either enable or disable these gestures. For single tap and double tap controls, see [Listeners](doc:listeners).

## Pan
```swift
mapView.mapOptions.isPanEnabled = true
```

## Pinch (Zoom)
```swift
mapView.mapOptions.isPinchEnabled = true
```

## Rotate
```swift
mapView.mapOptions.isRotateEnabled = true
```

## Tilt
```swift
mapView.mapOptions.isTiltEnabled = true
```

## All Gestures
```swift
mapView.mapOptions.setGesturesEnabled(true)
```
