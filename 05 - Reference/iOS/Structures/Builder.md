## Builder
`Builder` structure for [MFTLatLngBounds](ref:mftlatlngbounds).

```swift
Builder()
```
Method	|Description
--|--
build(): LatLngBounds|Builds MFTLatLngBounds with the current list of points.
include(latLng: LatLng)|Adds a point to the builder to be used in assessing the fit bounds of the MFTLatLngBounds.
