# DivIcon
### mapfit.DivIcon

Create a custom HTML element as an icon. You can then add this icon to a marker.
Constructor | Description
------------|------------
mapfit.DivIcon() | Create an instance of the custom HTML element with DivIcon. When an instance is created you will pass in an object that takes the following: icon size, anchor point, html as a string, and class name.
