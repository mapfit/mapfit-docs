# LatLngBounds
### mapfit.LatLngBounds

Constructor | Description
------------|------------
mapfit.LatLngBounds(sw: LatLng, ne: LatLng)| Constructs a rectangular bounding object from the provided south-west and north-east geographical coordinate values.

Method | Description
-------|------------
getCenter(): LatLng| Returns the center coordinates of the LatLngBounds. 
getNorthEast(): LatLng| Returns the north-east coordinates of the LatLngBounds. 
getNorthWest(): LatLng | Returns the north-west coordinates of the LatLngBounds. 
getSouthEast(): LatLng | Returns the south-east coordinates of the LatLngBounds. 
getSouthWest(): LatLng| Returns the south-west coordinates of the LatLngBounds.
