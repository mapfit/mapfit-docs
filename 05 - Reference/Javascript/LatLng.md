# LatLng
### mapfit.LatLng

Constructor | Description
------------|------------
mapfit.LatLng(latitude:Number, longitude:Number)| Creates an object representing a geographical point with the input latitude and longitude.

Property | Description
---------|------------
lat: Number| Latitude in degrees (WGS84). 
lng: Number| Longitude in degrees (WGS84).
