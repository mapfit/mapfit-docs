## Anchor
### com.mapfit.android.annotations.Anchor

`Anchor` enumerates available anchor points for  [Marker](ref:marker-1) icon. Top of the screen is considered as `Bottom` because the top-left point of the screen is (0,0).
```java
enum class Anchor
```
|Enum
--|--
|TOP
|TOP_LEFT
|TOP_RIGHT
|LEFT
|RIGHT
|CENTER
|BOTTOM
|BOTTOM_LEFT
|BOTTOM_RIGHT
