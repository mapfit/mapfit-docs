## OnMapThemeLoadListener
### com.mapfit.android.OnMapThemeLoadListener
Callback for listening to map scene load events after changing map theme or updating the scene values.
```java
interface OnMapThemeLoadListener
```

Method	|Description
--|--
onLoaded()|Called when the theme or scene is updated successfully.
onError()|Called when theme or scene is failed to update.
